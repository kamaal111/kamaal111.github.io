    ## Sessions

    Start a new session.

    <CopyableCode>
    ```bash
    tmux
    ```
    </CopyableCode>

    List sessions.

    <CopyableCode>
    ```bash
    tmux ls
    ```
    </CopyableCode>

    Attach to last session.

    <CopyableCode>
    ```bash
    tmux a
    ```
    </CopyableCode>

    Attach to a session using the session name.

    <CopyableCode>
    ```bash
    tmux a -t <SESSION_NAME>
    ```
    </CopyableCode>

    ## Windows

    Create window.

    ```
    (CTRL + B) C
    ```

    Next window.

    ```
    (CTRL + B) N
    ```

    Next window.

    ```
    (CTRL + B) P
    ```

    Switch/select window by number.

    ```
    (CTRL + B) 0...9
    ```

    ## Panes

    Split pane with horizontal layout.

    ```
    (CTRL + B) %
    ```

    Split pane with vertical layout.

    ```
    (CTRL + B) ""
    ```

    Switch to pane to the direction.

    ```
    (CTRL + B) ↑
    (CTRL + B) ↓
    (CTRL + B) →
    (CTRL + B) ←
    ```
