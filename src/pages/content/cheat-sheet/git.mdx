<Page title="Git">

    Stage changes by chunks.

    <CopyableCode>
        ```bash
        git add -p
        ```
    </CopyableCode>

    To discard current local changes.

    <CopyableCode>
        ```bash
        git restore .
        ```
    </CopyableCode>

    Remove cached git tracked files.

    <CopyableCode>
        ```bash
        git rm -r --cached .
        ```
    </CopyableCode>

    Cancel rebase.

    <CopyableCode>
        ```bash
        git rebase --abort
        ```
    </CopyableCode>

</Page>
